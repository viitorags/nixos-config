## <span style="color:#d3869b">Keybinds</span>
> <span style="color:#d8a657"><strong>Note:</strong></span> <span style="color:#d4be98">$mainMod</span> is set to the <span style="color:#a9b665">SUPER</span> key (usually the Windows key).

|<span style="color:#a9b665">Shortcut</span>                     | <span style="color:#d4be98">Description</span>                                             |                           
| ---------------------------  | ------------------------------------------------------- |
|<span style="color:#7daea3">$mainMod + Return</span>          | Opens the terminal                                      |
|<span style="color:#7daea3">$mainMod + Shift + Return</span>  | Opens a floating terminal with custom position and size |
|<span style="color:#7daea3">$mainMod + Q</span>              | Closes the active window                                |
|<span style="color:#7daea3">$mainMod + Shift + M</span>      | Exits Hyprland                                          |
|<span style="color:#7daea3">$mainMod + E</span>              | Opens the file manager (Yazi)                           |
|<span style="color:#7daea3">$mainMod + T</span>              | Toggles the window between floating and tiled           |
|<span style="color:#7daea3">$mainMod + R</span>              | Toggles fullscreen                                      |
|<span style="color:#7daea3">$mainMod + A</span>              | Opens the launcher/menu (Rofi)                          |
|<span style="color:#7daea3">$mainMod + L</span>               | Runs the custom power menu (Rofi)                       |
|<span style="color:#7daea3">$mainMod + P</span>               | Toggles pseudo-tile mode (dwindle)                      |
|<span style="color:#7daea3">$mainMod + J</span>               | Toggles horizontal/vertical split (dwindle)             |
|<span style="color:#7daea3">$mainMod + F</span>               | Opens the default browser (Brave)                       |
|<span style="color:#7daea3">$mainMod + Shift + F</span>       | Opens the browser in incognito mode                     |
|<span style="color:#7daea3">$mainMod + C</span>               | Opens the default editor (Neovim)                       |
|<span style="color:#7daea3">$mainMod + Shift + C</span>       | Opens the editor in a floating window                   |
|<span style="color:#7daea3">$mainMod + W</span>               | Changes the wallpaper (Rofi)                            |
|<span style="color:#7daea3">$mainMod + B</span>               | Restarts Waybar                                         |
|<span style="color:#7daea3">$mainMod + V</span>              | Manages clipboard (Rofi)                                |
|<span style="color:#7daea3">Insert</span>                     | Opens screenshot menu (Rofi)                            |
|<span style="color:#7daea3">Ctrl + Shift + Insert</span>      | Takes a screenshot of a selected area with grim+slurp   |
